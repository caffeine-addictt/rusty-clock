# Quick Start

### 1. Clone the repository

```sh
git clone https://github.com/caffeine-addictt/rusty-clock
cd rusty-clock
```

### 2. Build rusty-clock

```sh
cargo build
```

### 3. Run the rusty-clock

```sh
rusty-clock
```
